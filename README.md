# 🎬 YouTube → Insightful Article, PDF & Webpage Generator

A production-ready GenAI application that transforms YouTube videos into structured articles, downloadable PDFs, and fully styled webpages.

---

## 🚀 Overview

This project automates the process of converting YouTube video content into high-quality written formats using LLMs.

Instead of simple summarization, it performs:
- Transcript extraction
- Content cleaning
- Insightful article generation
- PDF export
- Webpage generation (HTML/CSS/JS)

---

## 🧠 Key Features

- 🎥 Extracts transcript from any YouTube video  
- ✍️ Converts raw transcript into a structured, professional article  
- 🧾 Generates downloadable PDF  
- 🌐 Builds a responsive webpage (HTML, CSS, JS)  
- ⚡ Interactive UI using Streamlit  
- 🔐 Secure API key input
